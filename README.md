# FEM Simulation Portfolio

## Overview

This repository contains a comprehensive collection of Finite Element Method (FEM) projects implemented using:

- MATLAB
- Python
- ANSYS Mechanical

The portfolio focuses on:
- Computational Mechanics
- Structural Analysis
- Nonlinear FEM
- Contact Mechanics
- Hyperelasticity
- Dynamic Analysis
- Parallel FEM Computing

The goal of this repository is to develop and demonstrate strong understanding of:
- FEM theory
- Numerical methods
- Scientific programming
- Industrial simulation workflows
- Verification and validation techniques

# Topics Covered

## Linear FEM
- 1D beam elements
- 2D elasticity
- Cantilever beam analysis
- Plane stress problems

## Nonlinear FEM
- Geometric nonlinearity
- Newton-Raphson solver
- Total and Updated Lagrangian formulation

## Material Nonlinearity
- Hyperelasticity
- Neo-Hookean material model

## Contact Mechanics
- Penalty-based contact
- Rigid surface interaction
- Contact stiffness formulation

## Stress Recovery
- Nodal Point Force (NPF) stress recovery
- Improved stress calculation methods

## Dynamic Analysis
- Modal analysis
- Harmonic response
- Frequency response analysis

## Thermal Analysis
- Steady-state thermal FEM
- Thermo-mechanical coupling

## High Performance Computing
- Parallel FEM assembly
- MPI-based element computations
- Scalability studies
